# GraphvizFrameworkPkg

The graphviz library components built into an XCFramework and wrapped up in a Swift Package

Based on graphviz 7.0.6

Includes:
    * lib/cdt
    * lib/cgraph
    * lib/common
    * lib/dotgen
    * lib/gvc
    * lib/label
    * lib/ortho
    * lib/pack
    * lib/pathplan
    * lib/xdot
    * plugin/core
    * plugin/dot_layout
    * plugin/quartz
    * dot_layout

    
Binary Frameworks for the following operating systems:
    * iOS
    * macOS
    * iOS Simulator
